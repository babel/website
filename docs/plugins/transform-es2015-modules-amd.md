This plugin transforms ES2015 modules to AMD.

## Installation

```sh
$ npm install babel-plugin-transform-es2015-modules-amd
```

## Usage

Add the following line to your `.babelrc` file:

```json
{
  "plugins": ["transform-es2015-modules-amd"]
}
```
