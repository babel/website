This plugin transforms ES2015 modules to SystemJS.

## Installation

```sh
$ npm install babel-plugin-transform-es2015-modules-systemjs
```

## Usage

Add the following line to your `.babelrc` file:

```json
{
  "plugins": ["transform-es2015-modules-systemjs"]
}
```
