This plugin transforms ES2015 modules to UMD.

## Installation

```sh
$ npm install babel-plugin-transform-es2015-modules-umd
```

## Usage

Add the following line to your `.babelrc` file:

```json
{
  "plugins": ["transform-es2015-modules-umd"]
}
```
